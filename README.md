# Geth Explorer

An EVM compatible explorer to learn about ethereum, solidity, etherjs, reactjs and more!

# Installation

```
git clone git@github.com:ragueneau/etherscan.git
```
```
cd etherscan
npm install
sudo npm install -g npm@8.8.0
```
## Usage
```
npm run start
```
